Projetos

Jogo em C

Este projeto acadêmico consiste no desenvolvimento de um jogo 2D em linguagem C utilizando a biblioteca Allegro 5. Inspirado em Chain Blaster, o jogo desafia o jogador a eliminar todos os inimigos criando campos de força que provocam reações em cadeia. A aplicação conta com menu inicial, três níveis de dificuldade progressiva, efeitos sonoros e interface gráfica, proporcionando uma experiência interativa. Durante o desenvolvimento foram aplicados conceitos de lógica de programação, estruturas de controle, organização de código e manipulação de eventos, consolidando os conhecimentos adquiridos ao longo da disciplina.
